### 数据来源

- 股票行情：baostock，后复权，日度（2020-01-01至2026-04-07）
- 市场指数：
  - 沪深300（000300）：作为CAPM分析的市场基准
  - 中证500（000905）：代表中小盘股票表现，用于对比分析
- 宏观指标：
  - CPI同比增速：来源国家统计局，反映通胀水平
  - M2同比增速：来源央行，反映流动性状况，选择理由：M2增速与股市流动性密切相关
- 财务数据：baostock，ROE和净利润率

### 存储方式

- 基础：CSV（方式 A）
- 进阶：SQLite（方式 C）
- 选择进阶方式的理由：SQLite适合关系型数据的存储与查询，便于跨表JOIN操作，适合金融数据分析场景

### 如何运行

1. 安装依赖：`pip install -r requirements.txt`
2. 运行 `01_download.ipynb` 下载原始数据
3. 运行 `02_clean.ipynb` 清洗并存储数据
4. 运行 `03_analysis.ipynb` 查看分析结果
5. 打开 `report.html` 阅读完整报告
6. （可选）渲染Quarto电子书：`quarto render`
